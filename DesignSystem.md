# Design System

This is a starter design system for the React app. The Figma prototype link is not publicly readable from this environment, so these values are a first-pass baseline based on the screenshot you shared and should still be verified against the design file in Figma.

## Primary Colors

- Ink: `#4A3B2D`
- Sand: `#F7EBDD`
- Paper: `#FFFDF9`
- Accent: `#F56E10`
- Border: `#E8CFA9`
- Muted Text: `#7A6A59`

## Fonts

- Headings: `Cormorant Garamond`, serif
- Body: `Inter`, sans-serif
- Fallback: `system-ui`, sans-serif

## Spacing Scale

- `8px`
- `16px`
- `24px`
- `32px`
- `48px`
- `64px`
- `96px`

## Usage Notes

- Use Ink for body text and darker supporting UI.
- Use Sand for warm section backgrounds.
- Use Accent for buttons, active nav states, and emphasis.
- Keep headings in the serif display family and body text in Inter.
- Keep layout spacing on a consistent rhythm so the page stays calm and easy to scan.
